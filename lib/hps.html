命令名称： copyContent - 读取到剪切板
命令功能： 从指定文件中复制内容到剪切板（会将 | 替换为换行符）
命令参数：
参数1 字符串型，文本路径
返回值： 整型，成功抓取了多少行文本
脚本例子： 
Msgbox lib.hps.copyContent("D:\DATA\log.txt")
命令名称： repeatDetection - 重复检测
命令功能： 检测参数1是否在参数2文件中有重复
命令参数：
参数1 字符串型，待检测的字符串
参数2 字符串型，文件保存路径
返回值：
有重复-false 
无重复-true
脚本例子： 
repeatReault = lib.hps.repeatDetection("testStr", "D:\DATA\log.txt")
命令名称： waitColorShow - 等待颜色出现
命令功能： 等待一个颜色出现
命令参数：
参数1 整型，检测位置x坐标
参数2 整型，检测位置y坐标
参数3 字符串型，判断的颜色
返回值： 无返回值
脚本例子： 
Call lib.hps.waitColorShow(100, 100, "#FFFFFF")
命令名称： waitColorHide - 等待颜色消失
命令功能： 等待一个颜色消失
命令参数：
参数1 整型，检测位置x坐标
参数2 整型，检测位置y坐标
参数3 字符串型，判断的颜色
返回值： 无返回值
脚本例子： 
Call lib.hps.waitColorHide(100, 100, "#FFFFFF")
命令名称： getRand - 获取随机数
命令功能： 获取随机整数
命令参数：
参数1 整型，随机数下限（包含
参数2 整型，随机数上限（包含）
返回值： 无返回值
脚本例子： 
randNum = lib.hps.getRand(1, 5)
命令名称： deleteFileInFolder - 删除所有文件
命令功能：
删除指定文件夹下的所有文件
注意：该方法在删除数量大时具有可见延迟
命令参数：
参数1 字符串型，需要进行内容删除的指定文件夹路径
返回值： 无返回值
脚本例子： 
Call lib.hps.deleteFileInFolder("D:\DATA")
命令名称： pasteTo - 粘贴到
命令功能： 在指定位置粘贴文本
命令参数：
参数1 整型，检测位置x坐标
参数2 整型，检测位置y坐标
参数3 字符串型，要粘贴的文本
返回值： 无返回值
脚本例子： 
Call lib.hps.pasteTo(100, 100, "testStr")
命令名称： paste - 直接粘贴
命令功能： 直接粘贴文本
命令参数：
参数1 字符串型，要粘贴的文本
返回值： 无返回值
脚本例子： 
Call lib.hps.paste("testStr")
命令名称： createNewFolder - 新建文件夹
命令功能： 新建文件夹，允许文件路径中存在尚未创建的文件夹
命令参数：
参数1 字符串型，需要新建的文件夹路径
返回值： true-新建成功 false-新建失败
脚本例子： 
Call lib.hps.createNewFolder("D:\new1\new2\new3")
命令名称： createNewFile - 新建文件
命令功能： 新建文件，允许文件路径中存在尚未创建的文件夹
命令参数：
参数1 字符串型，需要新建的文件路径
返回值： true-新建成功 false-新建失败
脚本例子： 
Call lib.hps.createNewFolder("D:\new1\new2\log.txt")
命令名称： save - 保存
命令功能： 保存数据
命令参数：
参数1 字符串型，键名
参数2 任意，保存的值
参数3 字符串型，要保存的文件夹路径
返回值： true-保存成功 false-保存失败
脚本例子： 
Call lib.hps.save("key", "value", "D:\new1\new2")
命令名称： load - 读取
命令功能： 读取数据
命令参数：
参数1 字符串型，键名
参数2 字符串型，保存到的文件夹路径
返回值： 字符串型，读取到的数据
脚本例子： 
value = lib.hps.load("key", "D:\new1\new2")
命令名称： setFocus - 窗口获取焦点
命令功能： 使用窗口名让指定窗口获取焦点（窗口名称请使用抓抓的句柄查找功能获取）
命令参数：
参数1 字符串型，窗口名
返回值： 无
脚本例子： 
Call lib.hps.setFocus("窗口名 - UC浏览器")
命令名称： colorShow - 颜色判断
命令功能： 判断指定点是否出现给定的多个颜色
命令参数：
参数1 整型，指定点x坐标
参数2 整型，指定点y坐标
参数3 数组，给定的多个颜色
返回值： true-颜色出现 false-颜色没出现
脚本例子： 
colors = Array("FFFFF0", "FFFF00", "FFF000")
result = lib.hps.colorShow(1, 1, colors)
命令名称： selectPic - 选择图片
命令功能： 打开选择图片对话框并选择图片
命令参数：
参数1 整型，能打开选择图片对话框的x坐标
参数2 整型，能打开选择图片对话框的y坐标
参数3 字符串，图片地址
返回值： 无
脚本例子： 
Call Lib.hps.selectPic(100, 100, "D:\DATA\pic.jpg")
